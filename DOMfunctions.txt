Нода (Node) - элемент структуры DOM-дерева отождествленный элементу 
структуры HTML разметки (<tag> ... </tag>, либо <tag />) и его содержимому.

Если нода находится в DOM-дереве HTML страницы, 
то при изменении ноды меняется и разметка в соответствии со структурой DOM-дерева.

Нода тега является родителем нод всех вложенных (1 уровня) тегов. Они, соответственно, называются ее детьми
Информация о родителе(ParentNode) и детях(ChildNodes) хранится в каждой ноде.

Порядок в списке детей в родительской ноде определяется порядком напсания тегов в ее HTML разметке. И наоборот.


elem.remove() - удаляет ноду elem из DOM дерева.

elem.append() - добавляет/перемещает в конец списка детей elem.

elem.prepend() - добавляет/перемещает в начало списка детей elem.

elem.before(elem1) - добавляет/перемещает elem1 перед elem.

elem.after(elem1) - добавляет/перемещает elem1 после elem.

elem.replaceWith(rElem) - удаляет elem из DOM дерева, вместо него вставляет rElem

elem.cloneNode() - возвращает elem без вложенных нод.

elem.cloneNode(true) - возвращает elem со всей вложеннной структурой нод.

elem.textContent - переменная, динамически сохраняющая внутренний текст без тегов elem.

elem.innerHTML - переменная, динамически сохраняющая HTML-разметку внутри тега ноды. 
  (Если изменить, то перепишется структура DOM внутри всей ноды)

document.createElement(tag) - возвращает пустую ноду разметки с тэгом tag не привязанной к DOM-дереву документа

elem.insertAdjacentText(*place*, text)/ elem.insertAdjacentHTML(*place*, HTML)/ elem.insertAdjacentElement(*place*, iElem) 
- Вставляет текст (text)/ разметку(HTML) / ноду по метке *place* не изменяя окружение: 
  'beforebegin' - перед elem в списке детей родительской ноды
  'afterend' -  после elem в списке детей родительской ноды
  'afterbegin' - в начало в списке детей elem
  'beforeend' - в конец в списке детей elem
